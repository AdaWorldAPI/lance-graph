## Key distinctions to preserve

### Anatomy vs field vs sweep vs bus

- anatomy = stored operator families and role-aware substrate
- field = searchable terrain / superpositional manifold
- sweep = cheap local collapse operator such as `bundle` or `xor_bind`
- bus = explicit structured reasoning transport

### Style is not metadata

Thinking style should be treated as collapse policy.
It lives in superposition inside `ResonanceDto` and collapses into bus settings inside `BusDto`.

### Contradiction is structure

Contradiction should not be treated as mere low confidence.
It should have an explicit representational home, likely in BRANCH and bus pressure fields.
